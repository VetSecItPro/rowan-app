# Rowan

> Family command center for modern households

**Live Site:** [https://rowanapp.com](https://rowanapp.com)

Rowan helps families organize tasks, meals, budgets, calendars, and goals in one beautiful app. Built for busy households that want to stay connected and organized.

## Tech Stack

- Next.js 15 / React 19
- TypeScript
- Supabase
- Vercel
